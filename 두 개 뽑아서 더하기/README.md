## 문제 설명
정수 배열 numbers가 주어집니다. numbers에서 서로 다른 인덱스에 있는 두 개의 수를 뽑아 더해서 만들 수 있는 모든 수를<br>
배열에 오름차순으로 담아 return하도록 solution 함수를 완성해주세요

numbers<br>
[2,1,3,4,1]<br>

result<br>
[2,3,4,5,6,7]

## 어려웠던 점
배열의 두 숫자 뽑아 더해서 배열에 담는건 쉬운일이다 하지만 중복을 허용하지 않는다는 점

## 새로 배운 점
두 가지 방법을 생각했다. 하나는 최소값을 스캔하고 최소값만 빼고 새 배열에 넣는 법<br>
또 하나는 최소값의 인덱스부터 왼쪽으로 밀어버리는 것이다.
