## 문제 설명
정수를 저장한 배열, arr에서 가장 작은 수를 제거한 배열을 리턴하는 함수 solution을 완성해주세요<br>
단, 리턴하려는 배열이 비었을 경우 -1을 채워 리턴하세요<br>

arr<br>
[4,3,2,1]<br>

return<br>
[4,3,2]

## 어려웠던 점
만약 리턴하려는 배열이 원래 모습을 유지하지 않아도 된다고하면 array.sort로 정렬한 후 왼쪽으로 한칸씩 밀어버리면 되는데<br>
이 문제에서는 원래 배열의 모습을 유지하고 있다.

## 새로 배운 점
두 가지 방법을 생각했다. 하나는 최소값을 스캔하고 최소값만 빼고 새 배열에 넣는 법<br>
또 하나는 최소값의 인덱스부터 왼쪽으로 밀어버리는 것이다.
