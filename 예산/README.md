## 문제 설명
물품을 구매할 총 예산을 입력받습니다. 각 부서가 금액을 신청하고 최대 몇 개의 부서에 물품을 지원할 수 있는지 return 하도록 solution 함수를 완성해주세요.
입출력 예시
d	budget	result
[1,3,2,5,4]	9	3
[2,2,3,3]	10	4

## 어려웠던 점
주어진 t의 문자열들을 어떻게 p.length만큼 나눠서 검사할지<br>
arr[]에 t의 문자열을 p.length크기만큼 나누어 넣어서 p와 arr을 대소비교를 해야한다고 생각했다.
