## 문제 설명
물품을 구매할 총 예산(budget)을 입력받습니다. 각 부서가 금액(d = [])을 신청하고 최대 몇 개의 부서에 물품을 지원할 수 있는지 return 하도록 solution 함수를 완성해주세요.<br>
입출력 예시<br>
d : [1,3,2,5,4], budget : 9,	result : 3 <br>
d : [2,2,3,3], budget :	10, result :	4<br>
## 어려웠던 점
배열의 특정 요소들의 합과 budget의 일치를 판별할 수는 있지만 어떻게 최대 지원 부서의 수를 세야 하는지를 모르겠다

## 새로 배운 점
그리디 알고리즘은 자료구조의 그래프에서만 쓰는게 아니라는 점
